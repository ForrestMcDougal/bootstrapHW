About the Data

Data was obtained from the sportradar MLB API.

  • 33 Postseason Games (2018)
  • 28 Umpires
  • 4,688 pitches called

The data retrieved from the API was:

  • Umpire name
  • Game ID
  • Home/Away team at bat
  • Called ball/strike
  • Pitch X location
  • Pitch Y location
  • Pitcher hand
  • Batter hand
  • Pitch type
  • Pitch speed
