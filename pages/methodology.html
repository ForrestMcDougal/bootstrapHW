Methodology

There are three overall questions that we wanted to anwer in this study:

  1. How accurate are umpires overall?
  2. Do players influence the accuracy of the umpires?
  3. Are there environmental influences on the umpires accuracy?

In order to answer these questions, data was obtained from the sportradar MLB API. First, the schedule for the 2018 MLB postseason was pulled down. For the games that were played (not all game IDs actually corresponed to a played game), the umpire's name was obtained from the game summary endpoint. The pitch data was then pulled for that same game for all pitches that were called a ball or strike by the umpire. The data was written to a csv file that could then be manipulated using pandas.

How accurate are the umpires overall?

This section was fairly straightforward. Using the location data and the call, statistics for each umpire and each pitch type could be calculated using pandas functionality. Accuracy (if the umpire called the theoretical call) was done by comparing the call to the location by adding a column to the DataFrame of theoretical calls and checking if the actual call and theoretical call were equivalent.

Do players influence the accuracy of the umpires?

This section had a similar process as the first, where there was more emphasis on pitch type and speed using pandas filtering and cutting to create smaller datasets.

Are there environmental influences on the umpires accuracy?

By environmental influences, we meant what the umpire was seeing (left vs right handed batter or home vs away at bat). This section had the biggest assumption that all missed calls benefited the pitching team because the overwhelming majority of missed calls were called a strike. In future work, we want to refine this to take into account balls vs strikes to improve the analysis.

About the Data

Data was obtained from the sportradar MLB API.

  • 33 Postseason Games (2018)
  • 28 Umpires
  • 4,688 pitches called

The data retrieved from the API was:

  • Umpire name
  • Game ID
  • Home/Away team at bat
  • Called ball/strike
  • Pitch X location
  • Pitch Y location
  • Pitcher hand
  • Batter hand
  • Pitch type
  • Pitch speed
